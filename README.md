# Jogo-de-castlevania-feito-totalmente-em-Java
Para jogar este jogo eu recomendo utilizar o netBeans, mas ele também pode ser utilizado no vsCode e outros editores de código compatíveis com Java.

Este jogo começou como um trabalho simples da faculdade, mas eu gostei tanto da idéia de desenvolver um jogo totalmente em Java que eu o desenvolvi muito mais, criando falas únicas para cada personagem e mecânicas de ataque para cada um deles.

Este jogo demorou extamente 1 semana para ser concluído.
